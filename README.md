## 🚀 Projects

### [Task Tracker](https://github.com/sumitjaiswar/task-tracker)
![Node.js](https://img.shields.io/badge/Node.js-339933?style=flat-square&logo=node.js&logoColor=white)
![TypeScript](https://img.shields.io/badge/TypeScript-3178C6?style=flat-square&logo=typescript&logoColor=white)
![PostgreSQL](https://img.shields.io/badge/PostgreSQL-316192?style=flat-square&logo=postgresql&logoColor=white)
![React](https://img.shields.io/badge/React-20232A?style=flat-square&logo=react&logoColor=61DAFB)

A full-stack task management app built with production-grade backend patterns.

- **Auth**: JWT refresh token rotation with reuse detection — compromised tokens invalidate the entire token family. Refresh tokens in HttpOnly cookies; access tokens in memory only.
- **Pagination**: Offset pagination for projects, cursor pagination for tasks (stable under concurrent writes).
- **Validation**: Zod schemas enforced at route level on request bodies and query params; mirrored on the client for consistent error handling.
- **Architecture**: Strict controller → service → repository separation; centralised global error handler; ownership-based access control.
